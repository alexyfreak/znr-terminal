  <read_first>
    RESEARCH.md Pattern 5 for Zustand store patterns
  </read_first>
  <action>
    Create store directory and 6 files:

    1. useSidebarStore.ts:
    - State: isExpanded (default false), isAnimating (default false)
    - Actions: toggle(), setExpanded(expanded), setAnimating(animating)
    - Persist with name 'zunoora-sidebar', storage createJSONStorage(() => localStorage)

    2. useThemeStore.ts:
    - Types: type Theme = 'dark' | 'light'
    - State: theme (default 'dark')
    - Actions: setTheme(theme), toggle()
    - Persist with name 'zunoora-theme'

    3. useLanguageStore.ts:
    - Types: type Language = 'uz' | 'ru' | 'en'
    - State: language (default 'uz')
    - Actions: setLanguage(language)
    - Persist with name 'zunoora-language'

    4. useSearchStore.ts:
    - State: query (default ''), isFocused (default false), isDocked (default false), results (default [] as string[])
    - Actions: setQuery(query), setFocused(focused), setDocked(docked), setResults(results), clear()
    - No persist needed (search state is ephemeral)

    5. useHistoryStore.ts:
    - State: items (default [] as Array<{id: string, title: string, type: string, date: string, docCount: number}>)
    - Actions: addItem(item), removeItem(id), setItems(items), clear()
    - Persist with name 'zunoora-history'

    6. useAccountStore.ts:
    - State: isLoggedIn (default false), userName (default ''), schoolName (default ''), role (default ''), avatar (default '')
    - Actions: login(userName, schoolName, role, avatar), logout(), updateProfile(data)
    - Persist with name 'zunoora-account'

    Each file: import { create } from 'zustand', import { persist, createJSONStorage } from 'zustand/middleware'. Use the exact pattern from RESEARCH.md Pattern 5.
  </action>
  <acceptance_criteria>
    - useSidebarStore.ts contains "isExpanded: false" and "persist"
    - useThemeStore.ts contains "theme: 'dark'" and "toggle"
    - useLanguageStore.ts contains "language: 'uz'" and "setLanguage"
    - useSearchStore.ts contains "query: ''" and "isDocked: false"
    - useHistoryStore.ts contains "addItem" and "persist"
    - useAccountStore.ts contains "isLoggedIn: false" and "logout"
    - All files export using "export const use*Store"
  </acceptance_criteria>
</task>
